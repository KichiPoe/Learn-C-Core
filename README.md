# About This Repository

This repository is my journey of learning **C programming from the ground up**.

The goal is not just to learn C syntax, but to understand how C actually works:

- How programs are compiled
- How memory works
- How data is stored
- How pointers interact with memory
- How manual memory management works
- How larger C projects are designed and organized

I am documenting everything I learn here:

- Concepts
- Experiments
- Code examples
- Projects
- Mistakes
- Debugging experiences
- Solutions and improvements

The idea is that this repository should remain useful not only for me in the future, but also for anyone who wants to learn **Core C** in a structured way.

---

# Why Learn C?

C is one of the most influential programming languages ever created.

Many technologies that power modern computing are built using C:

- Operating systems
- Databases
- Embedded systems
- Compilers
- Game engines
- Networking software

Learning C forces me to understand what is happening underneath the abstractions provided by higher-level languages.

This journey is about building that foundation.

---

# Learning Approach

I am following a project-based approach.

Instead of only reading concepts, every major topic will be connected with practical implementation.

The learning cycle:

```

Learn Concept
↓
Experiment
↓
Build Something
↓
Document Understanding
↓
Improve Existing Code

```

The focus is understanding **why things work**, not just making code compile.
